Uploading Resumes

Supported Formats

PDF Documents

Both digital and scanned PDFs are supported. Our OCR technology can extract text from image-based PDFs.

Multilingual Support

Upload resumes in English or Arabic. The system automatically translates and processes both languages.

Batch Upload

Upload multiple resumes at once (up to 16MB total). The system processes them automatically.

Best Practices for Resume Upload

  • Ensure PDFs are clear and readable - avoid heavily compressed or blurry scans
  • Use standard resume formats with clear sections (Experience, Education, Skills)
  • File names should be descriptive (e.g., "John_Doe_CV.pdf")
  • Remove password protection from PDFs before uploading
  • For best results, upload resumes that are 1-3 pages long

Frequently Asked Questions

How many resumes can I upload at once?
You can upload multiple resumes at once with a total file size limit of 16MB. There's no strict limit on the number of files, but we recommend uploading in batches of 10-20 for optimal performance.
Can I upload scanned PDFs?
Yes! Our system uses OCR (Optical Character Recognition) technology to extract text from scanned PDFs. Make sure the scans are clear and readable for best results.
How accurate is the AI matching?
Our AI matching system is highly accurate, using GPT-4o-mini with advanced prompt engineering techniques. The system provides detailed score breakdowns and confidence levels for transparency. We recommend reviewing the AI's analysis and using it as a decision-support tool.
What languages are supported?
The system currently supports English and Arabic resumes. Arabic content is automatically translated and transliterated to English for consistent processing and matching.
Can I export the matching results?
Yes! Click the "Export to PDF" button on the Results page to download a comprehensive report with all candidate scores, skills breakdown, and analysis.
How is my data protected?
We take privacy seriously. Personal information (names, emails, phone numbers) is automatically blurred in the interface. All data is processed securely, and we use encrypted connections to OpenAI's API.
What if the system rejects my resume?
The system validates uploaded files to ensure they are actual CVs/resumes. If a file is rejected, it may not contain enough CV-related keywords (experience, education, skills). Ensure you're uploading resume documents and not invoices, reports, or other file types.
How long does matching take?
The matching process typically takes 5-10 seconds per resume. For 10 resumes, expect around 1-2 minutes. The system processes candidates sequentially for maximum accuracy.
